Note: This version captures audio from your browser microphone, not from the server. All processing happens on the server.

Technical Details

Browser Compatibility

This application requires a modern browser with Web Audio API support:

  • Google Chrome 56+ (recommended)
  • Mozilla Firefox 52+
  • Microsoft Edge 79+
  • Safari 14.1+ (macOS/iOS)
